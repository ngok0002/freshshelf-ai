# FreshShelf AI

FreshShelf AI is a beginner-friendly full-stack JavaScript web application for food shelf-life estimation and storage recommendations.

The project is inspired by agentic AI workflows, but the implementation is intentionally simple and explainable. It uses a rule-based backend workflow to estimate remaining shelf life, evaluate spoilage risk, retrieve relevant storage knowledge, and generate practical recommendations.

## Features

- Product shelf-life estimation
- Storage condition simulation
- Risk level classification
- Risk factor explanation
- Recommendation generation
- Local knowledge assistant
- React dashboard UI
- Node.js/Express backend API

## Tech Stack

- Frontend: React, Vite, JavaScript, CSS
- Backend: Node.js, Express
- Data: Local JSON knowledge base
- Logic: Rule-based shelf-life estimation
- Search: Simple keyword-based retrieval
